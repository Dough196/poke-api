# poke-api

Technical challenge.

## Project Setup

### For frontend application

```sh
npm install
```

#### Compile and Hot-Reload for Development

```sh
npm run dev
```

### For backend application

```sh
cd server/
npm install
#copy or rename .env.example to .env
```

#### Compile and Hot-Reload for Development

```sh
#from server/
npm run dev
#from /
npm run start
```
